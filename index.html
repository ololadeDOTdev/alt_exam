FAQs

How does URL shortening work?

URL shortening works by taking a long URL and creating a shorter, condensed version that redirects to the original URL. When a user clicks on the shortened link, they are redirected to the intended destination.

Is it necessary to create an account to use the URL shortening service?

Yes it is necessary to create an account to use the url shortening service. The account allows us to better understand and serve you.

Are the shortened links permanent? Will they expire?

No, the shortened links are not permanent but they can be used for a long period of time.

Are there any limitations on the number of URLs I can shorten.

You can shorten as many URLs as you want.

Can I customize the shortened URLs to reflect my brand or content?

Yes, we have free design you can use to customize your URLs and also some special design that you can get after paying a token fee.

Can i track the performance of my shortened URLs?

Your shortened URLs' performance can be tracked anytime anywhere.

How secure is the URL shortening service? Are the shortened links protected against spam or malicious activity?

URL shortening service is 100% secure.

What is a QR code and what can it do?

A QR code (short for Quick Response code) is an array of black and white squares or pixels set in a grid that stores data for a machine to read.

Is there an API available for integrating the URL shortening service into my own applications or website?

Yes there is and for our valued customers it is free.
